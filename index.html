Tutorial: Using Timestamply by Decred

Introduction

Timestamply is a tool that proves the existence of something at a specific point in time. It provides decentralized and secure public proof of existence without the need for trusted authorities. Timestamply has various applications, including attestation, transparency in political processes, and legal contracts.

Steps to Use Timestamply

Step 1: Create a Timestamp

  • Go to timestamp.decred.org.
  • Drag and drop a digital file into the upload area.
  • The file generates a unique 32-byte string called a Merkle root or hash.
  • Keep the digital file unaltered after uploading.

Step 2: Anchor the Timestamp

  • The Merkle root is anchored to the blockchain on an hourly basis.
  • It takes approximately 15 minutes for the next anchor time.
  • After anchoring, it requires six block confirmations for the transaction to become irreversible.

Step 3: Monitor Timestamp Status

  • On the generate page, monitor the status and time indicator for completion.
  • Use the copy button and select button for the Merkle root.
  • The "Download Hash" button allows you to download the displayed hash.

Step 4: View Anchored Timestamps

  • Below the options, you'll find a list of the most recent anchored timestamps.
  • Use the search box to paste the Merkle root and search for individual proofs.
  • Details like Merkle Root and transaction hash can be used to query the Decred Blockchain.

Step 5: Verify Timestamp Authenticity

  • Once confirmations are received and the information is irreversible, the pending message is updated.
  • The "Download Proof" option becomes available, providing timestamp, Merkle Root, and transaction ID.
  • Copy the Merkle Root and go to the verify page.
  • Paste the Merkle Root in the provided box and click "Verify by Hash".

Step 6: Verify Document Authenticity

  • The verify log will return a clickable hash if the document is available.
  • Re-upload the document in the upload area and match it with the hash.
  • The verify log will confirm its authenticity with a clickable hash.
  • Use the hash to check the details of the timestamp on the Blockchain.

Conclusion

Congratulations! You have learned how to use Timestamply by Decred to upload and verify files, providing a timestamp and proof of existence at a specific point
